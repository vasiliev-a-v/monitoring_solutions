Программа пингует устройства (оборудование) по сети из списка в базе данных.
Программа завязана с "программой визуализации доступности сетевого оборудования" (ss_mon)

take_ping.sh
  - Записывает состояние оборудования (пропало/восстановилось)

msite_ping.sh
  - Отображает список оборудования в виде прямоугольников.
  - Если устройство в подсети не доступно, то прямоугольник красный.
  - Если устройтво доступно, то прямоугольник зелёный.
  - Запрашивает и отображает лог последних 10 событий пропадания/восстановления.

В базе данных ведётся хронологический журнал событий пропадания и восстановления устройств.

create_db_ping.sql
  - файл для создания базы
  - в конце файла закомментирован вариант её первичного заполнения (копирование из файла config.txt)

Файлы select_ping.sql и select_log.sql используются программой для запросов к базе.
Файл ping_triggers.sql - триггерная функция срабатывающая при изменении состояния оборудования. Записывает изменения в таблицу ping_changes


В crontab -e необходимо вставить:
#~ Периодический пинг устройств сети
  * *  *   *   *     sleep 30 && nice -n 19 /home/user/ping_equipment/take_ping.sh   &
  * *  *   *   *     sleep 40 && nice -n 19 /home/user/ping_equipment/msite_ping.sh  &
