В этом каталоге собраны программы осуществляющие контроль и управление различными устройствами [обезличенные прототипы]:
- asuv           - система управления вентиляцией [в серверной]
- dehydrator     - контроль состояния дегидратора (часы наработки и другие метрики)
- kbs_control    - централизованное управление контроллерами базовых станций Гудвин
- ping_equipment - контроль сетевой доступности оборудования
- ss_mon         - визуализация сетевой доступности оборудования
- take_electro   - контроль параметров по SNMP оборудования электроустановок Electro
- take_temp      - контроль температурного состояния в серверных
